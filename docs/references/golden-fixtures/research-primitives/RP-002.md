# RP-002 — Quantile Monotonicity Analysis

**Status:** active · v1 (2026-05-08)
**Category:** research-primitives
**Canonical:** `PythonDataService/app/research/validation/quantile.py::compute_quantile_analysis`

## What this pins

Quantile bin means, monotonicity ratio, and `is_monotonic` flag from `compute_quantile_analysis`, validated against direct `pd.qcut` + per-bin mean computation on 200 synthetic observations.

## Oracle

hand_computed — `pd.qcut(feature, n_bins, labels=False)` to assign bins, then `mean(returns[bin == b])` per bin. Monotonicity ratio = fraction of adjacent bin pairs where Q[b] < Q[b+1].

## Formula

```text
bins    = pd.qcut(feature, n_bins, labels=False)
bin_mean[b] = mean(returns where bins == b)

monotonicity_ratio = count(bin_mean[b+1] > bin_mean[b] for b in 0..N-2) / (N-1)
is_monotonic = (monotonicity_ratio == 1.0)
```

## Reference

Grinold & Kahn (1999) Ch. 6: quintile-spread analysis. A predictive signal produces monotonically increasing mean return across quantile bins.

## Tolerance

`atol=1e-9, rtol=0.0` — bin means are exact float64 arithmetic; monotonicity ratio is rational.
