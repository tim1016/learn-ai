## What this pins

Empirical p-value and percentile from `_empirical_position`, validated against the Phipson-Smyth formula applied directly to a seeded synthetic null distribution of 200 draws.

## Oracle

literature_formula — direct application of Phipson-Smyth (2010) Equation 2:

```text
p_value    = (1 + count(null >= observed)) / (N + 1)
percentile = mean(null < observed)
```

## Formula

The `+ 1` in numerator and denominator ensures p is never exactly 0 even when the observed value exceeds every null draw, reflecting the unknown probability of more extreme permutations.

## Reference

Phipson B & Smyth GK (2010). "Permutation P-values Should Never Be Zero: Calculating Exact P-values When Permutations Are Randomly Drawn." *Statistical Applications in Genetics and Molecular Biology* 9(1), Article 39.

## Tolerance
