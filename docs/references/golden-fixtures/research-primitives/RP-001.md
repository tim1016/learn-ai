# RP-001 — Information Coefficient (Spearman IC)

**Status:** active · v1 (2026-05-08)
**Category:** research-primitives
**Canonical:** `PythonDataService/app/research/validation/ic.py::compute_information_coefficient`

## What this pins

IC summary statistics — `mean_ic`, `t_stat`, `hit_rate`, and `daily_ic` array — from `compute_information_coefficient`, validated against direct `scipy.stats.spearmanr` calls on the same synthetic 5-day, 26-bar dataset.

## Oracle

literature_formula — `scipy.stats.spearmanr` per calendar day; mean IC, t-stat (ddof=1), and hit_rate computed from first principles without calling `compute_information_coefficient`.

## Formula

```text
daily_IC[d] = spearmanr(feature[day d], return[day d]).statistic
              (days with < 5 observations skipped)

mean_ic  = mean(daily_IC)
t_stat   = mean_ic / (std(daily_IC, ddof=1) / sqrt(N_days))
hit_rate = count(daily_IC * sign(mean_ic) > 0) / N_days
```

## Reference

Grinold & Kahn (1999) "Active Portfolio Management", Ch. 6. IC formula from Spearman rank correlation of factor signal vs realized next-period return.

## Tolerance

`atol=1e-9, rtol=0.0` — Spearman IC is computed identically by scipy; strict float64 arithmetic.
