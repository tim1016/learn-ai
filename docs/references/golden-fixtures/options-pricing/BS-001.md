# BS-001 — Black-Scholes European Call Price

**Status:** active · v1 (2026-05-08)
**Category:** options-pricing
**Canonical:** `PythonDataService/app/services/bs_greeks.py::bs_european_price`

## What this pins

The closed-form Black-Scholes-Merton call price `C = S·e^(-qT)·N(d1) - K·e^(-rT)·N(d2)` as implemented in `bs_european_price`, validated against py_vollib==1.0.1 as an independent oracle across a 180-case input grid.

## Oracle

`py_vollib.black_scholes.black_scholes(flag='c', S, K, t, r, sigma)` — GPL-licensed external library used for generation only, never imported in app/.

## Agreement

Max observed abs error: < 1e-14 (float64 accumulation noise).  
Pinned tolerance: `atol=1e-10, rtol=0.0`.  
The 1e-10 floor is conservative per `conventions.py` (1e-12 excluded: CI/dev platform divergence for transcendental functions).

## Input Grid

180 cases (5 spots × 3 strikes × 4 TTMs × 1 rate × 3 vols):

| Parameter | Values |
|-----------|--------|
| spot | 80, 90, 100, 110, 120 |
| strike | 90, 100, 110 |
| ttm_years | 1/12, 0.25, 0.5, 1.0 |
| rate | 0.05 |
| volatility | 0.15, 0.20, 0.30 |
| dividend | 0.0 |
| is_call | True |
