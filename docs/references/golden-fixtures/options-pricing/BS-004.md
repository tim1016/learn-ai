## What this pins

Call gamma `Γ = e^(-qT)·N'(d1) / (S·σ·√T)` from `black_scholes_greeks(...).gamma`, validated against py_vollib==1.0.1 across the same 180-case input grid as BS-001–003.

## Oracle

## Formula

```text
Gamma = e^(-qT) · N'(d1) / (S · σ · √T)

d1 = [ln(S/K) + (r - q + σ²/2)·T] / (σ·√T)
N'(x) = (1/√(2π)) · e^(-x²/2)  (standard normal PDF)
```
