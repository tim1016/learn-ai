# IV-001 — Implied Volatility Solver Round-Trip

**Status:** active · v1 (2026-05-08)
**Category:** options-pricing
**Canonical:** `PythonDataService/app/volatility/solver.py::implied_volatility`

## What this pins

The IV solver's convergence and numerical accuracy on a 12-case grid spanning ITM/ATM/OTM, short/long TTM, calls and puts, with and without dividend. For each case the `market_price` is the BSM price at `sigma_known`; the oracle is `sigma_known` itself — any convergent solver must recover it.

## Oracle

`sigma_oracle = sigma_known`. Market price generated from exact sigma via BSM; the oracle is the input sigma — no external library needed.

## Agreement

The solver's internal convergence criterion is 1e-7 (Newton-Raphson) or 1e-10 (Brent). Pinned tolerance: `atol=1e-6, rtol=0.0`. The 1e-6 floor provides one order of headroom.

## Input Grid

12 cases spanning:

| Parameter | Values |
|-----------|--------|
| spot | 100.0 |
| strike | 90, 100, 110 |
| ttm_years | 0.25, 1.00 |
| rate | 0.05, 0.02 |
| dividend | 0.00, 0.01 |
| sigma_known | 0.20, 0.25, 0.30, 0.35 |
| is_call | True (rows 0-8), False (rows 9-11) |

## Solver Cascade

Newton-Raphson → QuantLib → scipy Brent. All 12 cases must converge (status ∈ {NEWTON_OK, QUANTLIB_OK, BRENT_FALLBACK}).
