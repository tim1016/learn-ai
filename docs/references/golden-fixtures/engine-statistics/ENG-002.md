**Status:** active · v1 (2026-05-09)

## What this pins

The maximum drawdown formula `max_t[(peak_t - equity_t) / peak_t]` as implemented in `_max_drawdown`, validated against an independent numpy `maximum.accumulate` path across 3 synthetic 5-point equity curves.

## Oracle

hand_computed — `numpy.maximum.accumulate(curve)` for running peak, then vectorized `(running_max - equity) / running_max`. Independent from the canonical pure-Python loop with early-exit optimization.

## Formula

```text
max_drawdown = max over all t of (peak_t - equity_t) / peak_t
  where peak_t = max(equity[0..t])  and  peak_t > 0
```

Returns a positive fraction (0.20 = 20% drawdown).

## Reference

Bacon, *Practical Portfolio Performance Measurement and Attribution* (2e), §8.2.
