# ENG-001 — Sharpe Ratio (Daily Returns, Annualized)

**Status:** active · v1 (2026-05-08)
**Category:** engine-statistics
**Canonical:** `PythonDataService/app/engine/results/statistics.py::_sharpe`

## What this pins

The annualized Sharpe ratio formula `mean(r) / std(r, ddof=1) · √252` as implemented in `_sharpe`, validated against an independent numpy computation across 3 synthetic 5-element return series.

## Oracle

hand_computed — `numpy.mean` + `numpy.std(ddof=1)` annualized by `numpy.sqrt(252)`. Independent execution path from the canonical pure-Python loop.

## Formula

```
Sharpe = mean(r) / std(r, ddof=1) · √periods_per_year
```

where `ddof=1` is the sample standard deviation (N-1 denominator).

## Input Cases (hand-verifiable)

| Case | Returns | mean | var(ddof=1) |
|------|---------|------|-------------|
| 1 | [0.01, 0.02, -0.01, 0.03, 0.01] | 0.012 | 0.00022 |
| 2 | [0.005, 0.015, -0.005, 0.010, 0.000] | 0.005 | 0.0000625 |
| 3 | [-0.02, 0.04, -0.01, 0.02, -0.005] | 0.005 | 0.0006 |

## Agreement

Max observed abs error: < 1e-15.  
Pinned tolerance: `atol=1e-9, rtol=0.0`.

## Edge Cases (in test suite, not fixture)

- Zero std (all returns equal) → `_sharpe` returns `None`.
- Single return → `_sharpe` returns `None`.
