# ENG-004 — CAGR (Compound Annual Growth Rate)

**Status:** active · v1 (2026-05-09)
**Category:** engine-statistics
**Canonical:** `PythonDataService/app/engine/results/statistics.py::compute_portfolio_statistics`

## What this pins

CAGR from `compute_portfolio_statistics`, validated against an independent numpy power oracle across 3 synthetic single-year cases (trading_days=252).

## Oracle

hand_computed — `numpy.power(final / initial, 1 / years) - 1`, different floating-point path from canonical `(x)**(1/y)` in pure Python.

## Formula

```text
years = trading_days / 252
CAGR  = (final_equity / initial_cash)^(1/years) - 1
```

Canonical key line: `ann_return = (final_equity / initial_cash) ** (1 / years) - 1`

## Reference

Standard finance definition. See also Young (1991), "The Calmar Ratio: A Smoother Tool" (CAGR appears as numerator).

## Input Cases

| Case | initial | final | trading_days | CAGR |
|------|---------|-------|--------------|------|
| 1 | 1000 | 1100 | 252 | 0.100000 |
| 2 | 1000 | 1200 | 252 | 0.200000 |
| 3 | 1000 | 1150 | 252 | 0.150000 |

## Agreement

All cases: exact (years=1.0, so CAGR = final/initial − 1, no rounding).
Pinned tolerance: `atol=1e-9, rtol=0.0`.

## Additional Checks in Test Suite

- CAGR is positive when final > initial.
- years=0 guard: compute_portfolio_statistics returns 0.0 for zero-length periods.
