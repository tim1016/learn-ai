# ENG-005 — Calmar Ratio

**Status:** active · v1 (2026-05-09)
**Category:** engine-statistics
**Canonical:** `PythonDataService/app/engine/results/statistics.py::compute_portfolio_statistics`

## What this pins

Calmar ratio from `compute_portfolio_statistics`, validated against a composed oracle using the same numpy paths as ENG-002 (max drawdown) and ENG-004 (CAGR) across 3 synthetic cases.

## Oracle

hand_computed — `_oracle_cagr / _oracle_mdd` where each component uses the independent numpy path from its own fixture (ENG-002, ENG-004).

## Formula

```
years   = trading_days / 252
CAGR    = (final_equity / initial_cash)^(1/years) - 1
max_dd  = max_t[(peak_t - equity_t) / peak_t]
Calmar  = CAGR / max_dd
```

Canonical key lines: `calmar = ann_return / max_dd` (guarded: requires `max_dd > 0` and `years > 0`).

## Reference

Young, D. C. (1991). "The Calmar Ratio: A Smoother Tool." *Futures Magazine*.
