## What this pins

Per-horizon Spearman IC values from `compute_ic_decay_curve` (horizons 1..5), validated against direct `spearmanr` calls with overnight masking on 100 bars of synthetic 5-day data.

## Oracle

literature_formula — `scipy.stats.spearmanr` applied directly per horizon:

```text
for h in 1..max_horizon:
    for each trading day d:
        fwd_return[i] = log(close[i+h] / close[i])  if same calendar day else NaN
        IC_d[h] = spearmanr(ema[day_d], fwd_return[h][day_d]).statistic
    IC[h] = mean over days d of IC_d[h]
```

Overnight masking: bar pair (i, i+h) is excluded when any step in i..i+h-1 crosses midnight UTC. This prevents overnight gaps from contaminating intraday IC estimates.

## Reference

Grinold & Kahn (1999): IC decay quantifies signal persistence. A signal that predicts only 1-bar ahead will show rapid IC decay; a persistent signal maintains IC across horizons.

## Tolerance

`atol=1e-9, rtol=0.0` — Spearman IC computed identically by scipy with the same overnight masking logic.
