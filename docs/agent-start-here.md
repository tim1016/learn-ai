1. `AGENTS.md`
2. `docs/CURRENT.md`
3. The relevant stack rule in `.claude/rules/`

Then, for math or engine work, load both:

- `docs/math-sources-of-truth.md`

runtime artifacts, or local agent worktrees as implementation authority unless
`docs/doc-authority.md` says they are current.
