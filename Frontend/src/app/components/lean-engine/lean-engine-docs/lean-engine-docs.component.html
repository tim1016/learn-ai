Engine Docs

How the LEAN-compatible backtest engine turns raw minute bars into trade verdicts and summary statistics. Formulas match PythonDataService/app/engine/ line-for-line.

Engine Lab Strategy Docs

TL;DR

This engine reads zipped LEAN minute bars, consolidates them to the strategy's resolution, runs indicators in Decimal arithmetic, applies strategy rules, fills orders through a configurable fill model, tracks a portfolio, and emits per-trade and portfolio statistics — all with the invariant that the SPY EMA crossover reference algorithm produces exactly the same 63 trades, prices, indicator values, and net profit as LEAN's C# reference implementation.

New strategies don't need LEAN data — they just need to preserve the ordered WIN/LOSS sequence against a hermetic reference, which is the parity contract enforced by test_sma_crossover_parity.py.
