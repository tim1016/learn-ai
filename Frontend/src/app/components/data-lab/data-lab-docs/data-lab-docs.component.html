Guide

Pipeline overview

Data Lab generates a single CSV file containing minute-by-minute OHLCV data from Polygon.io with configurable technical indicators calculated by the pandas-ta library. A companion metadata JSON file documents every column.

Polygon.io REST API Fetches 1-minute OHLCV bars in ~111-day chunks (max 50,000 bars per request). Auto-paginates and de-duplicates. Prices are split/dividend-adjusted by default.
pandas-ta calculation Each selected indicator is applied to the full DataFrame. Column names follow pandas-ta conventions (e.g. EMA_10, MACD_12_26_9).
CSV + metadata export Single CSV file with unix_ts + iso_time as first two columns. Metadata JSON describes every column with source, library, and parameters.
