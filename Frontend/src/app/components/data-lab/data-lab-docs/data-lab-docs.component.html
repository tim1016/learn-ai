Overview

Data Lab generates a single CSV file containing minute-by-minute OHLCV data from Polygon.io with configurable technical indicators calculated by the pandas-ta library. A companion metadata JSON file documents every column.

Pipeline

Polygon.io REST API Fetches 1-minute OHLCV bars in ~111-day chunks (max 50,000 bars per request). Auto-paginates and de-duplicates. Prices are split/dividend-adjusted by default (configurable via the "Split & dividend adjusted" checkbox).
pandas-ta Calculation Each selected indicator is applied to the full DataFrame. Column names follow pandas-ta conventions (e.g. EMA_10, MACD_12_26_9).
CSV + Metadata Export Single CSV file with unix_ts + iso_time as first two columns. Metadata JSON describes every column with source, library, and parameters.

Split & Dividend Adjustment

By default, all OHLCV prices are adjusted for historical stock splits and dividend distributions (matching the Polygon.io API default of adjusted=true). This means historical prices are retroactively modified so that price series are continuous across corporate actions — a $100 stock that does a 2-for-1 split will show $50 for all pre-split bars.

Why this is the default: Adjusted prices produce continuous series that are essential for accurate technical indicator calculations (EMAs, RSI, MACD, etc.) and meaningful backtesting. Without adjustment, a split or dividend would appear as a sudden price gap, distorting every indicator that relies on price history.

When to use unadjusted data: Uncheck "Split & dividend adjusted" if you need raw exchange prices — for example, when verifying trade executions at historical prices, when your indicator logic handles adjustments separately, or when analysing the raw tape for microstructure research.

CSV Metadata File

The metadata JSON file accompanies the CSV and documents every column. Download it alongside the CSV using the "Metadata JSON" button in Data Lab.

Metadata Structure

dataset

Ticker, date range, timespan, bar count, generation timestamp

data_source

Polygon.io provider details, API version, chunking strategy

calculation_engine

Library name (pandas-ta), version, GitHub URL

columns[]

Array of every CSV column with: name, type, description, source/library, parameters used

notes[]

Precision, NaN handling, timestamp convention, de-duplication
