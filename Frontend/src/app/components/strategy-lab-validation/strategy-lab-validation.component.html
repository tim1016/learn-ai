Strategy Lab Validation

Compare our backtest results against an external source (e.g. TradingView)

How to use this page

Quick start

  1. Export OHLCV bars from TradingView using the Pine Script below.
  2. Upload the OHLCV bars CSV — this feeds the exact same price data into our engine.
  3. Optionally upload the Trade Report CSV (Strategy Tester → Export → Trade List) for side-by-side comparison.
  4. Set the same RSI parameters (Window=14, Oversold=30, Overbought=70).
  5. Click Run Backtest.

Data source modes

  • Import CSV Bars — Runs our strategy on TradingView's exported bars. This eliminates price data differences and isolates pure strategy logic comparison.
  • Use Polygon Data — Runs on our own Polygon.io data. Useful for general backtesting but prices will differ from TradingView.

Strategy: RSI Reversal

Always-in-the-market RSI strategy matching TradingView's built-in "RSI Strategy":

  • RsiLE — RSI crosses below oversold level → close short, open long
  • RsiSE — RSI crosses above overbought level → close long, open short
