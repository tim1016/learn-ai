Overview

This pipeline cleans raw Polygon.io minute-level OHLCV data to produce reliable, indicator-ready datasets. It addresses fabricated bars on early-close days, fractional volumes, misleading cumulative VWAP, and opaque indicator warmup — all of which distort EMA, MACD, RSI, and ADX calculations and cause TradingView mismatch.
