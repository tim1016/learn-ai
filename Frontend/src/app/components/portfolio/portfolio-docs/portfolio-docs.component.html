When Does Drift Occur? Drift can happen if a trade is manually edited or deleted, if the position engine has a bug that's later fixed, or if concurrent operations cause a race condition. The reconciliation system acts as the self-healing layer.

Data Model

Core entities and their relationships in the portfolio system.

Entity Key Fields Role
Account Name, Type, Cash, InitialCash Trading account (Paper or Backtest)
PortfolioTrade Side, Quantity, Price, Fees, Multiplier Immutable trade record (source of truth)
Position NetQuantity, AvgCostBasis, RealizedPnL, Status Derived aggregate of open lots per ticker
PositionLot EntryPrice, Quantity, RemainingQuantity Individual FIFO lot created by a buy trade
OptionLeg EntryIV, EntryDelta, EntryGamma, EntryTheta, EntryVega Greeks snapshot at trade time for risk calculations
PortfolioSnapshot Equity, Cash, MarketValue, Greeks Point-in-time portfolio valuation for equity curves
RiskRule RuleType, Threshold, Action, Severity Configurable risk limit with Warn/Block action
StrategyTradeLink TradeId, StrategyExecutionId Links trades to strategy executions for attribution
