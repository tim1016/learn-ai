Linear Approximation — Known Limitation Scenario analysis uses entry Greeks (delta, vega, theta) for first-order approximation. Entry Greeks drift as price, time, and IV change — they are snapshots, not live values. For large moves (>5% price shock or >10% IV change), second-order effects (gamma, vanna, volga) may cause significant deviation from actual P&L. A full-revaluation mode (repricing via Black-Scholes) would eliminate this approximation error but is not yet implemented.
